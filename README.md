## Requirements

* SilverStripe ^4.0
* SilverStripe Elemental ^2.0

## Installation

`composer require dynamic/silverstripe-elemental-accordion`

## Usage

Elemental Accordion Block will add the following Element to your site:

* Accordion (collapsable panels)
